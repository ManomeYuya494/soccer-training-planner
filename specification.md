# サッカートレーニング計画・効率化ツール 仕様書 (最速MVP版)

## 1. プロジェクト概要
サッカー指導者の最大の負担である「今日のトレーニング内容を考え、A4用紙に清書する」作業を、AIの提案とPDF自動生成により最速で完了させる。

## 2. 開発フェーズ (スモールスタート)

### Phase 1: コア機能（ここから着手）
「AIが考え、綺麗なA4になる」という一連の流れを最速で実現する。
- **入力フォーム**: 
    - 参加人数、時間、場所、テーマ、指導者のこだわり（前提条件）。
- **AIメニュー提案 (o1-mini/GPT-4o)**:
    - 具体的数値（グリッドサイズ、配置、ルール）を含む3つの案を提示。
- **固定テンプレートPDF出力**:
    - 選んだ案を、そのまま現場で使えるプロ品質のA4レイアウト（PDF）に自動変換。
    - ※この時点では図解はテキストまたは簡易的な説明のみ。

### Phase 2: 現場テスト後の拡張（必要に応じて追加）
Phase 1を実際に現場で使いながら、本当に必要なものだけを追加する。
- **簡易図解エディタ**: 配置図をポチポチと作成できる機能。
- **履歴・振り返り機能**: 過去の計画を保存し、一言メモを残せる機能。
- **ダッシュボード**: 練習の積み上げの可視化。

## 3. 技術スタック
- **Frontend**: Next.js, Tailwind CSS
- **AI**: OpenAI API (最新モデル)
- **PDF生成**: react-pdf 等

## 4. 目指すゴール
「スマホやPCでテーマを入れるだけで、5分後には練習メニューが印字されたA4の紙が手元にある」状態を作る。
